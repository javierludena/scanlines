# Roadmap

This roadmap describes the current public direction for Scanlines. It is not a commitment. Priorities may change based on product needs, technical constraints, and user feedback.

## Available now

- Android app
- Video game collection tracking
- Platform-based organization
- Physical copy details
- Edition, condition, region, and notes
- Visual collection browsing

## In progress

- Better catalog coverage
- Barcode and catalog improvements
- Wishlist improvements
- More platform data
- Collection browsing refinements
- Collection import and export

## Planned

- iOS version
- Advanced statistics
- Expanded platform support
- Improved catalog correction flow
- Official scanlines.app email addresses for support and security

## Under consideration

- Social features
- Public collection sharing
- More collector-focused discovery tools
- Additional metadata for editions, barcodes, variants, and regional releases
