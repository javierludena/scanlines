# Scanlines

Scanlines is a visual collection app for video game collectors.

It helps collectors track physical and digital game collections, organize games by platform, record physical copy details, and browse collections visually.

The Android app is available on Google Play:

- Website: https://scanlines.app
- Android: https://play.google.com/store/apps/details?id=app.scanlines

Scanlines is in active development.

## Product focus

- Track video game collections
- Organize games by platform
- Support physical copy details
- Record edition, condition, region, and notes
- Browse collections through a visual interface
