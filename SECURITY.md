# Security

Please report security issues privately.

Do not open a public GitHub issue for security problems.

Security contact:

- askjato@gmail.com

## Scope

Security reports may include:

- Authentication issues
- Data exposure
- Account access problems
- API vulnerabilities
- Privacy-related bugs

Include a clear description, reproduction steps if available, affected account or device details if relevant, and any screenshots or logs that help explain the issue.
